Présent -> Imparfait

1. appeler: Présent: Nous ... ; Imparfait: Tu ..., vous ...
gọi: Hiện tại: Chúng tôi... ; Quá khứ tiếp diễn: Bạn..., bạn (số nhiều/lịch sự)...
2. jeter: Présent: Nous ... ; Imparfait: Je ..., ils ...
ném: Hiện tại: Chúng tôi... ; Quá khứ tiếp diễn: Tôi..., họ...
3. geler: Présent: Nous ... ; Imparfait: Il ..., nous ...
đóng băng: Hiện tại: Chúng tôi... ; Quá khứ tiếp diễn: Trời..., chúng tôi...
4. emmener: Présent: Nous ... ; Imparfait: Tu ..., ils ...
dẫn đi: Hiện tại: Chúng tôi... ; Quá khứ tiếp diễn: Bạn..., họ...
5. répéter: Présent: Nous ... ; Imparfait: Elle ..., vous ...
lặp lại: Hiện tại: Chúng tôi... ; Quá khứ tiếp diễn: Cô ấy..., bạn (số nhiều/lịch sự)...

Bạn đã trả lời đúng 0 trên tổng số 0 câu.
