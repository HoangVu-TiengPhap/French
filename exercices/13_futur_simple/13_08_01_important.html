Groupe 3 (Futur Simple)

1. (aller) Je ..., vous ...
(đi) Tôi sẽ..., bạn (số nhiều/lịch sự) sẽ...
2. (faire) Tu ..., ils ...
(làm) Bạn sẽ..., họ sẽ...
3. (savoir) Nous ..., elles ...
(biết) Chúng tôi sẽ..., họ (nữ) sẽ...
4. (cueillir) Il ..., elles ...
(hái, nhặt) Anh ấy sẽ..., họ (nữ) sẽ...
5. (refaire) Je ..., vous ...
(làm lại) Tôi sẽ..., bạn (số nhiều/lịch sự) sẽ...

Bạn đã trả lời đúng 0 trên tổng số 0 câu.
